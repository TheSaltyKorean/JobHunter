ATS Login Credentials
Stored locally in your browser. Used to auto-fill sign-in and create-account pages on Workday, Greenhouse, Taleo, and other ATS sites. Your email doubles as the login for most sites.
How it works: When Auto-Fill encounters a login/create-account page, it fills your email (or username on Taleo) and password. If you've previously applied on a site, it logs you in. If it's a new account, it fills the registration form with these credentials.

Security note: Credentials are stored in Chrome's local extension storage (never synced or sent anywhere).

How to start: Run the companion server from your extension folder:

The server wraps your local claude CLI — no API key needed, uses your existing Claude subscription. Without the server running, unrecognized fields are simply skipped.
